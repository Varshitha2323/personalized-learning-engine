# Personalized Learning Decision System

## Problem
Self-studying students struggle with deciding what to study next, planning realistically,
and tracking progress toward their goals.

## Solution
A backend-driven system that makes explainable study decisions, adapts plans based on
student behavior, and tracks consistency and progress.

## MVP Features
- User authentication
- Study goal and topic management
- Study decision engine
- Time tracking
- Progress analytics

## Tech Stack
- FastAPI
- PostgreSQL
- SQLAlchemy
- Streamlit (frontend)
